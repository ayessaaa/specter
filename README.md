# specter

A game about a knight bravely trying to escape the cave but he is haunted by his own ghost. His last gameplay shows up as a ghost in the next run. Would he let this distract him or use it for his advantage? 

**It's for you to decide.**

try it here: https://shaaarkai.itch.io/specter

### Controls:
- **W** = Jump
- **A** = Left
- **D** = Right
- **SPACE** = Dash
- **ESC** = Menu
